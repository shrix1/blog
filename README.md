## TECH STACK 
 - Stack : `NextJS13` `TailwindCSS`
 - Database : `MongoDB`
 - Auth : `Next-auth`
 - Images/svg : `Popsy`
 - Icons : `React-Icons`
 - Hosting : `Vercel`

---
## Setup
1. Fork and Clone the rep0
2. run `npm install`
3. setup the `.env` file
4. run `npm run dev`
5. Create, a new Branch, make code change make a PR
6. HAPPY CODING!
